# Mail-X

# Tech Stack
- NodeJS
- Express
- MongoDB
- ReactJs

Deployed on Heroku.

# Running the Backend

## Build and run
  1.Clone Repo:

    git clone https://github.com/steverogers07/Mail-X.git
    
  2.To install all the dependencies, enter command:
    
    npm install
    
  3.To run:
    
    npm run dev


# Running Frontend

## Build and run
    
  1.To install all the dependencies, enter command:
    
    npm install
    
  2.To run:
    
    npm start
